# V2.0 Alpha 3 验收记录

版本：`2.0.0a7`

## 验收目标

本阶段验收 PIT 固定候选进入现有严格账本后的工程正确性和归因完整性，不验收真实收益目标，也不修改生产默认。

必须满足：

- 生产默认离线黄金结果与 v1.6 完全一致；
- Alpha 3 候选不能从 YAML 选为生产策略；
- 四臂共享股票过滤、组合和成交实现；
- 覆盖匹配价格臂单独隔离 PIT 缺失筛选效应；
- 影子执行分数禁止携带任何未来收益标签；
- 未来 PIT 数据扰动不能改变过去分数；
- Alpha 2 封存、固定候选定义和双数据指纹必须通过前置校验；
- 四臂订单/成交/公司行动和汇总结果全部封存并通过严格 SHA256 校验；
- 合成数据只能得到 `nonproduction_data_only`。

## 建议验证命令

```bash
env -u UV_DEFAULT_INDEX uv lock --check --default-index https://pypi.org/simple
env -u UV_DEFAULT_INDEX uv sync --locked --extra public --default-index https://pypi.org/simple
uv run --no-sync ruff check src tests
uv run --no-sync python -W error::FutureWarning -m unittest discover -s tests -v
uv run --no-sync python run.py demo --seed 7 --output results/demo_v2_alpha3
uv run --no-sync python run.py result-verify \
  --output results/demo_v2_alpha3 --strict
```

真实 PIT 验收和 Alpha3 运行：

```bash
uv run --no-sync python run.py pit-verify --config config.yaml
uv run --no-sync python run.py pit-research \
  --config config.yaml \
  --output results/pit_factor_research_v2_alpha2
uv run --no-sync python run.py result-verify \
  --output results/pit_factor_research_v2_alpha2 --strict
uv run --no-sync python run.py pit-shadow \
  --config config.yaml \
  --alpha2-research results/pit_factor_research_v2_alpha2 \
  --output results/pit_shadow_v2_alpha3
uv run --no-sync python run.py result-verify \
  --output results/pit_shadow_v2_alpha3 --strict
```

## 当前环境边界

云端没有用户 Tushare Token 和真实 PIT 缓存。合成数据验收只证明防前视、四臂严格账本、归因和封存正确；不能填写真实覆盖率、收益增量或前瞻模拟晋级结论。
