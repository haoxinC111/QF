# v1.5 Alpha 增强与预声明协议

> v1.5.1 状态更新：公开历史回放没有形成一致 Alpha 增量，严格通道生产默认已恢复为 `legacy_v1_4`。本文保留 v1.5 候选在结果生成前冻结的公式与协议；其当前状态为 `experimental`，默认晋级决定为 `rejected`。治理依据见 [`V1.5.1_GOVERNANCE.md`](V1.5.1_GOVERNANCE.md)。

## 目标与边界

v1.5 引入冻结的 `quality_momentum_v1_5` 实验候选。目标是检验能否提高动量信号的质量和下行风险调整后表现，而不是用历史数据搜索一组恰好达到 15% 年化的参数。项目不承诺收益；2013–2025 的市场结果已经在旧版本研究中被查看，任何 v1.5 回放都只能称为历史检验。

本页中的公式和权重在生成任何 v1.5 新回测结果前冻结。代码不包含自动寻优、网格搜索或根据 2022–2025 表现切换参数的逻辑。

## 因子定义

所有价格因子使用截至信号日可得的总回报价格。12–1 月形成期收益记为：

\[
PRET_t=P_{t-21}/P_{t-252}-1
\]

FIP 信息离散度采用 Da、Gurun 与 Warachka 的定义。在同一形成期的 231 个日收益中，令 `%pos` 和 `%neg` 分别为正、负收益日占全部有效日的比例：

\[
ID_t=sign(PRET_t)\times(\%neg-\%pos)
\]

低 `ID` 表示收益由较多连续小幅变动形成，高 `ID` 表示收益集中于少数跳变。长多策略使用交互项：

\[
FIPMom_t=PRET_t\times(1-ID_t)
\]

这样，累计涨幅相同的股票中，连续形成的赢家高于少数跳涨形成的赢家；连续下跌的输家不会因为单独的“平滑度”得分而被错误奖励。论文原始定义见 [Da, Gurun & Warachka (2014), *The Review of Financial Studies*](https://doi.org/10.1093/rfs/hhu003)。

另外加入两个只使用历史价格的防御特征：

- `low_downside_vol`：60 日下行波动率的相反数；下行波动率为负收益平方均值开根号后年化。
- `drawdown_quality`：当前总回报价格相对过去 126 个交易日最高价的回撤，数值越接近 0 越好。

在 v1.5 候选中，总波动率仍用于逆波动率定仓，不再同时在选股分数里重复加权。20 日平均成交额继续作为硬性可交易门槛，不再把“大成交额”当成 Alpha。

## 冻结实验候选权重

| 因子 | 权重 |
| --- | ---: |
| 12–1 动量 | 25% |
| 6–1 动量 | 15% |
| FIP 连续信息动量 | 25% |
| 200 日长期趋势 | 10% |
| 低下行波动 | 15% |
| 126 日回撤质量 | 10% |
| 总波动率选股项 | 0% |
| 流动性选股项 | 0% |

严格通道沿用 v1.4 的动态成分、行业/市值中性化、排名缓冲、行业与单股上限以及风险开关。公开通道保留六组 v1.3 旧方案，并新增一个 `quality_momentum_v1_5` 候选；该候选与 `momentum_focus_15` 使用相同的 15 股、退出排名、单股上限和风险仓位参数，以尽量隔离 Alpha 变化。

## 验收协议

1. 单元测试证明修改信号日之后的价格不会改变当日全部新因子或选择。
2. 构造累计收益相同的“连续上涨”和“单日跳涨”路径，前者必须具有更低 `ID` 和更高 `FIPMom`。
3. 严格与公开通道调用同一个特征实现，避免公式漂移。
4. 严格研究套件消融当前配置的正权重因子；公开稳健性检查对 v1.5 候选逐因子删除并运行 5/10/20 bps 成本压力。
5. 离线 demo、完整单元测试、静态检查和结果产物校验必须通过。
6. 只有绑定相同公开数据指纹、由同一次研究运行生成的旧/新策略结果才允许比较。仓库中旧 `results/` 数值不得当作 v1.5 实证结果。
7. 2013–2025 全部标为已查看历史数据；真正的前瞻证据来自冻结代码后的 2026+ 新数据或模拟盘。

年化 15% 不是验收门槛。历史年化达到 15% 但在分期、消融或成本压力下失效的方案不会被视为成熟；反之，未达到 15% 也不会通过反复调参伪装成满足目标。
