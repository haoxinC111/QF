## 4. 本地全量验收清单

等待下载 Agent 完成 B3 后，在隔离集成目录执行：

```bash
env -u UV_DEFAULT_INDEX uv sync --locked --extra public \
  --default-index https://pypi.org/simple

uv run --no-sync ruff check src tests
uv run --no-sync python -W error::FutureWarning \
  -m unittest discover -s tests

uv run --no-sync python run.py validate-data --config config.yaml

uv run --no-sync python run.py pit-lake-build \
  --config config.yaml \
  --archive-root /绝对路径/data_lake

uv run --no-sync python run.py pit-lake-verify \
  --config config.yaml \
  --archive-root /绝对路径/data_lake
```

必须检查：

1. `research_eligible=true`；
2. `archive_bridge.mode=strict`；
3. B0/B1/B3 三个 snapshot 与本地 pass decision 完全一致；
4. `source_files_verified` 等于 `selected_task_count`；
5. 财报和估值覆盖率均不低于配置的 `minimum_symbol_coverage`；
6. `pit-snapshot` 在公告当日不可见、下一交易日可见；
7. 只有上述条件全部满足后，才运行 Alpha2/Alpha3。
